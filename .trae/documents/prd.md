## 1. 产品概述

律师事务所内部案件材料管理系统，用于集中管理案件相关的文件和材料，通过树形结构直观展示案件材料的层级关系，支持材料的增删改查及导出清单功能，提升律师团队对案件材料的组织效率与协作能力。

- 目标用户：律师事务所内部律师及助理人员
- 核心价值：解决案件材料分散、查找困难、清单整理耗时等问题，实现材料管理的标准化与数字化

## 2. 核心功能

### 2.1 用户角色

| 角色 | 说明 | 核心权限 |
|------|------|----------|
| 律师 | 主办律师 | 创建/编辑/删除案件及材料，导出清单 |
| 助理 | 律师助理 | 查看案件材料，协助编辑材料，导出清单 |

### 2.2 功能模块

1. **案件列表页**：案件搜索与筛选，案件卡片展示，新建案件入口
2. **案件材料树管理页**：材料树形结构展示与操作，材料详情查看与编辑，拖拽排序
3. **导出清单页**：清单模板选择，导出预览与生成，支持 PDF/Excel 导出

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 案件列表页 | 搜索栏 | 按案件名称、编号、状态搜索 |
| 案件列表页 | 筛选器 | 按案件类型、承办律师、创建时间筛选 |
| 案件列表页 | 案件卡片 | 显示案件名称、编号、类型、状态、材料数量、最近更新时间 |
| 案件列表页 | 新建案件 | 弹窗填写案件基本信息（名称、编号、类型、描述） |
| 案件材料树管理页 | 材料树 | 树形结构展示案件材料，支持展开/折叠，显示文件类型图标 |
| 案件材料树管理页 | 节点操作 | 添加子文件夹/文件、重命名、删除、移动（拖拽） |
| 案件材料树管理页 | 材料详情面板 | 右侧面板显示选中材料的详细信息（名称、类型、大小、上传时间、备注） |
| 案件材料树管理页 | 材料编辑 | 编辑材料备注、上传文件替换 |
| 导出清单页 | 模板选择 | 选择预设清单模板（诉讼材料清单、证据目录、卷宗目录等） |
| 导出清单页 | 材料勾选 | 勾选需要导出的材料项，支持全选/反选 |
| 导出清单页 | 预览区域 | 实时预览导出清单的排版效果 |
| 导出清单页 | 导出操作 | 生成 PDF 或 Excel 文件下载 |

## 3. 核心流程

**案件材料管理主流程**：用户进入系统 → 浏览案件列表 → 选择案件进入材料树页面 → 对材料进行增删改查操作 → 选择材料导出清单

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深藏青色 (#1B2A4A)，传达专业、严谨的法律行业气质
- **辅助色**：暖金色 (#C9A96E)，用于重要操作按钮和强调元素，体现品质感
- **背景色**：浅灰白 (#F5F6FA)，舒适不刺眼，适合长时间使用
- **卡片/面板**：白色 (#FFFFFF)，带微妙阴影，层次分明
- **按钮风格**：圆角矩形 (8px)，主要按钮实色填充，次要按钮边框样式
- **字体**：标题使用 Noto Serif SC（宋体风格，庄重正式），正文使用 Noto Sans SC（清晰易读）
- **布局风格**：左侧导航 + 顶部工具栏 + 内容区，卡片式布局
- **图标风格**：线性图标，笔画统一，使用 Lucide 图标库

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 案件列表页 | 搜索栏 | 圆角输入框，搜索图标，深色边框聚焦态 |
| 案件列表页 | 筛选器 | 下拉选择器，标签式多选，淡色背景 |
| 案件列表页 | 案件卡片 | 白色卡片，左侧色条标识状态，hover 提升阴影 |
| 案件材料树管理页 | 材料树 | 左侧树形面板，展开/折叠箭头，文件类型图标，选中高亮 |
| 案件材料树管理页 | 详情面板 | 右侧滑出面板，表单字段，操作按钮组 |
| 案件材料树管理页 | 工具栏 | 顶部操作栏，图标+文字按钮，添加/删除/导出 |
| 导出清单页 | 模板选择 | 卡片式模板选择，选中态金色边框 |
| 导出清单页 | 材料勾选 | 复选框树形列表，全选/反选快捷按钮 |
| 导出清单页 | 预览区域 | A4 纸张样式预览，分页线，打印效果模拟 |
| 导出清单页 | 导出按钮 | 金色主按钮，下拉选择格式 |

### 4.3 响应式设计

- 桌面优先设计，适配 1280px 及以上宽度
- 材料树页面左右分栏，最小宽度 1024px 时面板可折叠
- 不支持移动端（内部系统以 PC 为主）
